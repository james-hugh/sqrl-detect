## 2026-02-19 - Enhanced CLI Visibility and Accessibility
**Learning:** In CLI environments, pairing ANSI color codes with explicit textual labels (e.g., [SECURE], [CRITICAL]) significantly improves both glanceability for power users and accessibility for users with color vision deficiencies or non-color terminals.
**Action:** Always include textual state indicators alongside color-coded UI elements in terminal applications.

## 2026-03-20 - Visual Hierarchy and Iconography in Menus
**Learning:** Organizing CLI menus with color-coded headers (Yellow) and pairing actions with themed emojis (e.g., 🕹️ for Engage, 💀 for Exit) reduces cognitive load and creates a more engaging feel. Emojis serve as immediate visual anchors, making options easier to distinguish at a glance.
**Action:** Use themed iconography and distinct headers to improve the visual hierarchy of terminal-based menus.
