## 2026-02-19 - Enhanced CLI Visibility and Accessibility
**Learning:** In CLI environments, pairing ANSI color codes with explicit textual labels (e.g., [SECURE], [CRITICAL]) significantly improves both glanceability for power users and accessibility for users with color vision deficiencies or non-color terminals.
**Action:** Always include textual state indicators alongside color-coded UI elements in terminal applications.

## 2026-03-12 - Themed CLI Hierarchy and Iconography
**Learning:** Using themed iconography (emojis) paired with color-coded headers and numeric labels in CLI menus significantly reduces cognitive load and improves command discoverability. Centralizing these UI elements in macros ensures consistency and easier maintenance of the "brand" feel across different terminal functions.
**Action:** Use UI theming macros to pair emojis with actions and headers in CLI applications to create a cohesive and engaging user experience.
