## 2026-02-19 - Enhanced CLI Visibility and Accessibility
**Learning:** In CLI environments, pairing ANSI color codes with explicit textual labels (e.g., [SECURE], [CRITICAL]) significantly improves both glanceability for power users and accessibility for users with color vision deficiencies or non-color terminals.
**Action:** Always include textual state indicators alongside color-coded UI elements in terminal applications.

## 2026-03-12 - Visual Hierarchy and Menu Emotional Resonance
**Learning:** Organizing CLI menus with color-coded headers (Yellow) and pairing actions with themed emojis (🕹️, 💀) reduces cognitive load and creates a more engaging "premium" feel. Consistent use of macros for terminal control sequences (like screen clearing) ensures cross-function visual stability.
**Action:** Implement a clear visual hierarchy using color headers and thematic iconography for all main navigation points.
