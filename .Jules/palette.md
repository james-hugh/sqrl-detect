## 2026-02-19 - Enhanced CLI Visibility and Accessibility
**Learning:** In CLI environments, pairing ANSI color codes with explicit textual labels (e.g., [SECURE], [CRITICAL]) significantly improves both glanceability for power users and accessibility for users with color vision deficiencies or non-color terminals.
**Action:** Always include textual state indicators alongside color-coded UI elements in terminal applications.

## 2026-04-08 - Persistent CLI Command Centers and Gamification
**Learning:** Transforming linear CLI scripts into persistent Command Centers with loop-based menus and gamified feedback (like "Pillow Fort Integrity") increases user engagement and makes the tool feel more like a cohesive application. Case-insensitive input via `strcasestr` also significantly lowers the barrier for entry.
**Action:** Implement persistent loop menus and interactive state variables (integrity, progress) to enhance CLI application "feel".
